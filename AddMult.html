Add Left to Right


Often, math problems are written horizontally on job applications. This is because the employer wants you to see that you can do the math in your head.

The following short video (on the right) demonstrates how to solve two-digit addition problems from left-to-right starting with the highest place-value, rather than solving the problem right-to-left, which involves the need to work it out on paper and carrying to the next place value.

Mental Multiplication

There are several ways to mentally compute multi-digit multiplication problems in your head, depending on the type or complexity of the problem.

For Example: Knowing that multiplying any number by 0 results in 0 helps you to multiply any numbers that end with 0 quickly.

(If this confuses you, Remember: the multiplication symbol: x, stands for the word "of.")

So a multiplication sentence such as "3 x 6" can be read as, "Three groups of six."

Likewise, 0 X 42 can be read as, "Zero groups of forty-two."

Using these terms clearly demonstrate that zero groups of any number will always be zero (and vice-versa).

A problem like: 120 x 60 can be done quickly in your head as follows:

1. Eliminate the 0s (Only if they are at the end!)

2. Solve the rest of the problem: 12 x 6 = 72.

3. Put the # of 0's you took out back in the answer at the end:

120 x 60 = 7200

Multiplication is Repeated Addition

When you multiply two numbers, you are really just adding them repeatedly. For Example: 3 x 4 is the same as "Three groups of four." It looks like this: 4 + 4 + 4 = 12). It is much easier to do this: 3 x 4 = 12.
