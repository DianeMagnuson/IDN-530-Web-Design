Think Backwards: Use Addition


Do you remember learning "Fact Families?"
Fact Families consist of 3 numbers. The same three number can be rearranged to make inverse (opposite) operations true.

For Example: Let's look at these 3 numbers: 6, 4, and 10.

A true math sentence for these three numbers is: 6 + 4 = 10.

The inverse of this math sentence is also true: 10 - 6 = 4.

Subtraction is the opposite of addition.

Now, let's say that we have this problem: 96 - 78 = ?

Let's use the Think Backwards: Use Addition! strategy:

Think: 78 + ? = 96

By reversing the problem, I create an addition problem which is mentally easier to solve.

All I need to do now is count up from 78 to see how much it takes to get to 96, like this:

Counting by 10's from 78, I get:
88, 98 (that's 20!) But the number I want to count up to is 2 digits less:

So, 20-2 = 18! The answer to 96 - 78 is 18!

Divisibility Tests!


Knowing the tests for divisibility for the numbers helps you know which numbers to use for dividing quickly. If you have forgotten them, or never learned them before, here they are:

A number is divisible by:
2 = if the last digit of the number is 0, 2, 4, 6, or 8.
3 = if the sum of the digits of the number is divisible by 3.
4 = if the last two digits of the number are divisible by 4.
5 = if the last digit of the number is 0 or 5.
6 = if the number is divisible by both 2 and 3.
7 = eliminate last digit, multiply last digit by 2, then subtract from remaining number
8 = if the last 3 digits are divisible by 8.
9 = if the sum of the digits is divisible by 9.
10 = if the last digit of the number is 0.
